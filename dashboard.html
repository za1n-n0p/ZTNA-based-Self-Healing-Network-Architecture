🔍 How AI Analysis Works
When a BLOCKED event is recorded, PDP sends full event context to Ollama qwen3:4b running locally on the Windows machine.
The model analyzes: IP, attack type, severity, trust score, block history, packet rate, zone, and action taken.
Output: threat level (CRITICAL/HIGH/MEDIUM/LOW), attack pattern classification, 2-sentence assessment, and a specific recommendation.

DDoS Correlation Engine
💣 How DDoS Detection Works
The DDoS Correlation Engine monitors all attack events in a 60-second sliding window. If 3 or more unique attacker IPs target the same victim IP within that window, the system automatically flags it as a Distributed Denial of Service (DDoS) attack. All participating IPs are blocked simultaneously. The event is logged to the database and displayed here.
